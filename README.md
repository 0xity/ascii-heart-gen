# ASCII Heart Generator
A python script that can generate ASCII hearts of any size.
It supports text input, spanning and repeating the text inside the heart.
The shape of the heart can be customized to some degree, with more freedom planned for the future.

> [!IMPORTANT]
> Recommended size range is anywhere between 2 and 12.

> [!WARNING]
> Sizes above 12 have an issue with extra characters being added on the sides.

This is the first project where I've used trigonometry to solve a problem.

Module support is planned for the future.
